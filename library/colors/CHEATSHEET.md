CML CHEATSHEET (short)
----------------------
File: color_master_library.sh  (v4.0.0)

Usage:
  source ~/kh-scripts/library/colors/color_master_library.sh
  colors                - show the palette
  cml_apply_theme NAME  - switch theme (classic|solarized|neon|darkwave|minimal)
  cml_info              - show info

Demo script:
  ~/kh-scripts/library/colors/colors-demo.sh

Notes:
  - This library provides helpers for 16-color, 256-color and TrueColor.
  - Use 'colors' to preview after switching themes.
